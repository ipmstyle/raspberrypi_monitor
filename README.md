# Raspberry Pi Monitor

## INSTALL

매 5분마다 라즈베리파이의 CPU/GPU 온도를 측정하여 로그로 저장한다.

'''
./temp_for_crontab.sh
'''

## CPU Temperature

1. 현재 온도 보기

```rpi_cpu_temp```

2. crontab

CPU 온도보기
sudo cat /var/log/cpu_temp

CPU 온도보기
sudo cat /var/log/cpu_temp

## GPU Temperature

1. 현재 온도 보기
