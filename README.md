# Raspberry Pi Monitor

## INSTALL

  매 5분마다 라즈베리파이의 CPU/GPU 온도를 측정하여 로그로 저장한다.

  `./temp_for_crontab.sh`


## 온도 히스토리

1. 현재 CPU 온도 보기

    `rpi_cpu_temp`
    
1. 현재 GPU 온도 보기

    `rpi_gpu_temp`

1. crontab 로그파일 보기

    CPU 온도 로그파일 보기<br />
    `sudo cat /var/log/cpu_temp`

    CPU 온도 로그파일 보기<br />
    `sudo cat /var/log/cpu_temp`
